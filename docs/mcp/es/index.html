Servidor MCP

Conecta tu asistente de IA directamente a los datos del RCSD — calendarios, menús del almuerzo, reuniones de la junta, matrícula y más.

Endpoint https://mcp.rcsd.info/mcp

¿Qué es MCP?

El Protocolo de Contexto de Modelos (MCP) es un estándar abierto que permite a los asistentes de IA llamar herramientas externas y acceder a datos en tiempo real. En lugar de depender únicamente de datos de entrenamiento, tu IA puede obtener información actualizada del servidor MCP del RCSD — como el menú del almuerzo de hoy o el resumen de la próxima reunión de la junta escolar.

El servidor MCP del RCSD es público y de uso gratuito. No requiere clave de API ni autenticación. Utiliza el transporte HTTP con transmisión continua (no SSE, no stdio), lo que significa que funciona como servidor remoto — no hay nada que instalar localmente.

Instalación

Claude Desktop

Abre Configuración → Desarrollador → Editar configuración para editar tu archivo claude_desktop_config.json y agrega el servidor rcsd:

{
  "mcpServers": {
    "rcsd": {
      "url": "https://mcp.rcsd.info/mcp"
    }
  }
}

Reinicia Claude Desktop. Las herramientas del RCSD deberían aparecer en el menú de herramientas (ícono de martillo).

Claude Code (CLI)

Agrega el servidor con un solo comando:

claude mcp add rcsd --transport http https://mcp.rcsd.info/mcp

O agrégalo al archivo .mcp.json de tu proyecto:

{
  "mcpServers": {
    "rcsd": {
      "type": "url",
      "url": "https://mcp.rcsd.info/mcp"
    }
  }
}

Claude.ai (Web)

Claude.ai admite servidores MCP remotos de forma nativa:

  1. Ve a claude.ai y abre Configuración (parte inferior izquierda)
  2. Navega a Integraciones
  3. Haz clic en Agregar más y elige Integración personalizada
  4. Ingresa https://mcp.rcsd.info/mcp como URL
  5. Nómbralo RCSD y guarda

Las herramientas del RCSD estarán disponibles en cualquier conversación nueva.

Cursor / VS Code / Windsurf

La mayoría de los editores compatibles con MCP admiten un archivo de configuración .mcp.json o mcp.json. Agrega lo siguiente a la raíz de tu proyecto o a la configuración de usuario:

{
  "mcpServers": {
    "rcsd": {
      "type": "url",
      "url": "https://mcp.rcsd.info/mcp"
    }
  }
}

ChatGPT

ChatGPT no admite servidores MCP de forma nativa en este momento. Sin embargo, puedes acceder a los mismos datos directamente en rcsd.info, o utilizar la API JSON pública si estás creando un GPT personalizado con acciones.

Otros clientes MCP

Cualquier cliente MCP que admita el transporte HTTP con transmisión continua puede conectarse. Apúntalo a:

https://mcp.rcsd.info/mcp

No se necesitan encabezados de autenticación. El servidor admite el protocolo HTTP con transmisión continua estándar de MCP. Consulta la documentación de MCP para detalles sobre implementación de clientes.

Herramientas disponibles

HerramientaDescripción
list-schools Lista las 12 escuelas del RCSD con nombre, grados, tipo y matrícula
query-school Información detallada de una escuela específica — horario de clases, director/a, dirección, PTO, enlace al menú del almuerzo y más. Acepta nombre o identificador (ej. "orion", "Roy Cloud").
check-calendar Verifica si hay clases en una fecha determinada, incluyendo días festivos, vacaciones, días de horario reducido y días regulares. Cubre los años escolares 2025-26 y 2026-27.
get-lunch-menu Obtiene el menú del almuerzo en tiempo real para una escuela en una fecha específica. Extrae datos en vivo de HealthePro y reconoce los días sin clases.
get-meeting-summary Resúmenes de reuniones de la junta — ya sean las reuniones más recientes o una fecha específica. Generados por IA a partir de transcripciones de reuniones.
get-meeting-details Detalles completos de una reunión específica de la junta — puntos de agenda, adjuntos con enlaces de descarga PDF, marcas de tiempo del video, marcadores de capítulo, enlace a la transcripción y URLs de origen (BoardDocs/Simbli/YouTube). Acepta una fecha (YYYY-MM-DD) o slug de la reunión.
get-school-board-items Puntos del orden del día de la junta que mencionan una escuela específica. Útil para rastrear decisiones que afectan a un plantel en particular.
get-sped-data Datos de matrícula de educación especial (IEP) — totales del distrito o desglose por escuela, incluyendo conteos por grado y colocación en el LRE.

Ejemplos de consultas

Una vez conectado, prueba hacerle preguntas a tu asistente de IA como:

  • "¿Qué hay de almuerzo en Orion hoy?"
  • "¿Hay clases el 31 de marzo de 2026?"
  • "Lista todas las escuelas del RCSD y su matrícula."
  • "¿Qué pasó en la última reunión de la junta escolar?"
  • "¿Qué se discutió en la reunión de la junta del 25 de marzo de 2026 y en qué momento del video?"
  • "Muéstrame el horario de clases de Roy Cloud."
  • "¿Cuáles son los números de educación especial en Clifford?"
  • "¿Qué puntos del orden del día de la junta han mencionado a Adelante?"

Fuentes de datos

Todos los datos del servidor MCP provienen de documentos públicos del RCSD. La información de las escuelas, los calendarios y las transcripciones de las reuniones de la junta se recopilan y mantienen en rcsd.info. Los menús del almuerzo se obtienen en tiempo real del sistema HealthePro utilizado por el distrito. El código fuente está disponible en GitHub.

Compilado de forma independiente a partir de documentos públicos del RCSD. Documentos fuente en rcsdk8.net y el portal GAMUT de la junta.
