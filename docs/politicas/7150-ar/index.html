7150 AR Reglamento Administrativo

Selección y desarrollo de terrenos

ÚLTIMO CAMBIO: 04/02/2026 · VERIFICADA: 24/5/2026 · ID DE REVISIÓN: JNccDQy9JH7cjSB45EqMHQ==
Versión oficial en Simbli ↗ Ver JSON ↗

Traducción automática (IA) para ayudarte a entender la política. La única versión oficial y con valor legal es la versión en inglés, en Simbli ↗

Como parte del proceso de selección de sitios del distrito, el Superintendente o su designado deberá: - Reunirse con las autoridades locales de recreación y parques que correspondan para revisar todos los métodos posibles de coordinar la planificación, el diseño y la construcción de nuevas instalaciones escolares y sitios escolares, o de ampliaciones importantes a instalaciones escolares existentes y a instalaciones de recreación y parques en la comunidad. (Education Code 35275) - Notificar por escrito a la agencia de planificación local que corresponda y solicitar su informe y recomendaciones sobre la conformidad del sitio propuesto o de la ampliación propuesta con el plan general adoptado. (Government Code 65402; Public Resources Code 21151.2) - Hacer que el sitio sea investigado por personal competente con respecto a las tendencias de población, el transporte, el suministro de agua, las instalaciones de eliminación de residuos, los servicios públicos, los peligros de tráfico, las condiciones de drenaje superficial y otros factores que afecten los costos iniciales y de operación. Esta investigación deberá incluir estudios geológicos y de ingeniería de suelos para evitar ubicar la escuela en terrenos que tengan el potencial de sufrir daños por terremotos u otros peligros geológicos según lo especificado en Government Code 65302. (Education Code 17212-17212.5) - Hacer una solicitud por escrito de información necesaria o útil para evaluar y determinar la seguridad de un sitio escolar propuesto, o de una ampliación a un sitio escolar existente, a una persona, corporación, empresa de servicios públicos, empresa de servicios públicos de propiedad pública local, o agencia gubernamental en lo que respecta a tuberías, líneas de transmisión y distribución eléctrica, ferrocarriles y tanques de almacenamiento, de conformidad con la ley. (Education Code 17212.2, 17251) - Asegurarse de que el sitio cumpla con los estándares estatales para la selección de sitios escolares según lo especificado en 5 CCR 14010-14012. - Garantizar el cumplimiento de la Ley de Calidad Ambiental de California (CEQA, por sus siglas en inglés) según lo exige la ley, incluyendo la publicación de los avisos requeridos en el sitio web del distrito. (Public Resources Code 21000-21177) - Notificar por escrito al Departamento de Educación de California antes de adquirir el título o arrendar el sitio si el sitio propuesto se encuentra dentro de dos millas en línea aérea de una pista de aeropuerto o de una pista propuesta. (Education Code 17215) - Realizar un análisis de calidad del aire conforme al Health and Safety Code 44360 y al Education Code 17213 si el sitio propuesto se encuentra dentro de 500 pies del borde del carril de tráfico más cercano de una autopista u otro corredor de tráfico de mucho flujo, y determinar que la calidad del aire en el sitio propuesto es tal que ni la exposición a corto ni a largo plazo representa riesgos significativos para la salud de los estudiantes. (Education Code 17213) En la selección y el desarrollo de proyectos financiados conforme al Programa de Instalaciones Escolares de 1998 (Proposición 1A) contenido en Education Code 17070.10-17077.10, el Superintendente o su designado deberá: - Determinar si el sitio propuesto está libre de contaminación tóxica asegurándose de que se realice una evaluación ambiental de Fase I y/o una evaluación preliminar de peligro según lo exige la ley. (Education Code 17213.1) El Superintendente o su designado deberá asegurarse de que la evaluación preliminar de peligro esté disponible para revisión y comentarios públicos de conformidad con Education Code 17213.1. - Presentar un informe de resumen anual de gastos a la Junta Estatal de Asignaciones de conformidad con la ley. (Education Code 17076.10) - Incluir en los planos una conexión cableada a una red telefónica pública conmutada o la utilización de tecnología inalámbrica. (Education Code 17077.10) - Establecer una meta de participación de al menos el tres por ciento, por año, del monto total en dólares gastado cada año por el distrito en empresas de veteranos con discapacidad. (Education Code 17076.11)
