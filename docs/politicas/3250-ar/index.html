3250 AR Reglamento Administrativo

Tarifas de transporte

ÚLTIMO CAMBIO: 24/05/2023 · VERIFICADA: 24/5/2026 · ID DE REVISIÓN: 4Z991d6ccf9nHcMtw5sJyQ==
Versión en inglés Ver JSON ↗

Traducción automática (IA) para ayudarte a entender la política. La única versión oficial y con valor legal es la versión en inglés, en Simbli ↗

Cuando la Mesa Directiva lo apruebe, el distrito podrá cobrar tarifas de transporte a los estudiantes que viajan hacia y desde la escuela. (Education Code 39807.5) Con la aprobación de la Mesa Directiva, el distrito también podrá cobrar tarifas de transporte por: Participantes en un programa comunitario de recreación ofrecido conforme a Education Code 10900-10914.5 (Education Code 10913, 39835) Estudiantes que viajan entre las escuelas regulares de tiempo completo a las que asisten y las clases ocupacionales regulares de tiempo completo proporcionadas por un centro o programa de ocupaciones regional (Education Code 39807.5) Estudiantes que viajan hacia y desde un lugar de empleo durante el verano en relación con un programa de empleo de verano para jóvenes (Education Code 39837) El monto total que el distrito reciba del estado y de las tarifas de los padres/tutores no deberá exceder el costo operativo real del transporte de casa a la escuela durante el año fiscal. Si se cobran tarifas en exceso debido a errores en los costos estimados, las tarifas se reducirán en los años siguientes. (Education Code 10913, 39801.5, 39809.5, 39837) Los pases y boletos de autobús se venderán de forma electrónica, cuando estén disponibles, y en todas las escuelas del distrito y en la oficina del distrito. No se cobrará dinero en los autobuses escolares.
