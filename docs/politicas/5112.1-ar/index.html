5112.1 AR Reglamento Administrativo

Exenciones de asistencia escolar

ÚLTIMO CAMBIO: 13/01/2010 · VERIFICADA: 24/5/2026 · ID DE REVISIÓN: OZubjFFPjYuAYzpTo7v4HA==
Versión oficial en Simbli ↗ Ver JSON ↗

Traducción automática (IA) para ayudarte a entender la política. La única versión oficial y con valor legal es la versión en inglés, en Simbli ↗

Un/a estudiante puede quedar exento/a de la asistencia escolar de tiempo completo en el programa de educación regular del distrito si: - Recibe instrucción en una escuela privada de tiempo completo y el supervisor de asistencia escolar del distrito o su designado/a verifica que la escuela privada ha presentado una declaración jurada conforme a Education Code 33190 (Education Code 48222, 48223) - Recibe instrucción de un/a tutor/a privado/a que posee una credencial estatal válida para el grado que imparte, siempre que la instrucción consista en estudio y recitación durante al menos tres horas diarias por 175 días de cada año calendario (Education Code 48224) - Posee un permiso de trabajo para trabajar temporalmente en la industria del entretenimiento o industrias afines (Education Code 48225) - Posee un permiso de trabajo y asiste a clases de tiempo parcial (Education Code 48230) - Tiene entre 12 y 14 años de edad e ingresa a una zona de asistencia escolar proveniente de otro estado dentro de los 10 días anteriores al final del período escolar, siendo aplicable la exención por el resto de dicho período (Education Code 48231)
