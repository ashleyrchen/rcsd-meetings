3541 AR Reglamento Administrativo

Rutas y servicios de transporte

ÚLTIMO CAMBIO: 04/11/2009 · VERIFICADA: 24/5/2026 · ID DE REVISIÓN: QHYCVufqhJJEssekTIEpDA==

Rutas y Paradas de Autobús El Superintendente o su designado diseñará las rutas y paradas de transporte para promover la seguridad de los estudiantes y el máximo aprovechamiento de los autobuses. Los estudiantes serán elegibles para el servicio de transporte hacia y desde la escuela si la distancia entre la parada de autobús establecida por la escuela y la escuela está más allá del mínimo indicado a continuación: - Para estudiantes de escuela primaria: Grados K-3: tres cuartos de milla Grados 4-8: una milla - Para estudiantes que asisten a una escuela secundaria de tres años: Grados 7-9: una milla - Para estudiantes que asisten a una preparatoria de cuatro años: Grados 9-12: dos millas El Superintendente o su designado podrá autorizar el transporte dentro de la distancia caminable cuando existan problemas o peligros de seguridad. Los estudiantes que asisten a una escuela fuera de su área de asistencia pueden ser elegibles para los servicios de transporte de acuerdo con la política de la Mesa Directiva. El Superintendente o su designado comunicará por escrito a los padres/tutores sobre las rutas, horarios y paradas de autobús y/o gestionará que los medios de comunicación locales publiquen dicha información. Servicios de Transporte Con la autorización de la Mesa Directiva de Educación, el distrito podrá proporcionar u organizar servicios de transporte para: - Estudiantes que viajan hacia y desde la escuela durante el día escolar regular (Education Code 39800) - Excursiones y viajes de campo (Education Code 35330) - Actividades escolares, exposiciones o ferias, u otras actividades determinadas como beneficiosas para los estudiantes (Education Code 39860) - Empleados del distrito y padres/tutores que viajan hacia y desde actividades educativas autorizadas por el distrito (Education Code 39837.5) - Estudiantes de preescolar o jardín de niños (Education Code 39800) - Estudiantes que viajan a clases ocupacionales de tiempo completo proporcionadas por un Programa o Centro de Ocupaciones Regionales (Education Code 39807.5, 41850) - Estudiantes que viajan hacia y desde sus lugares de empleo durante el verano en conexión con un programa de empleo de verano para jóvenes (Education Code 39837) - Adultos matriculados o inscritos que viajan hacia y desde la escuela, o adultos con fines educativos distintos a los traslados hacia y desde la escuela (Education Code 39801.5) - Estudiantes de escuelas privadas, de la misma manera y en las mismas rutas proporcionadas para los estudiantes del distrito (Education Code 39808) - Fines no escolares según lo permitido por la ley, tales como: - Recreación comunitaria (Education Code 39835) - Transporte público (Education Code 39841) - Transporte de empleados del gobierno hacia y desde sus lugares de empleo (Education Code 39840) El distrito proporcionará transporte del hogar a la escuela y servicios de transporte adicionales según sea necesario para los estudiantes con discapacidades, conforme a lo especificado en sus programas de educación individualizada. (Education Code 41850; 20 USC 1400-1482; 34 CFR 104.4) El Superintendente o su designado proporcionará transporte a los niños en situación de calle de acuerdo con la ley, la política de la Mesa Directiva y el reglamento administrativo.
