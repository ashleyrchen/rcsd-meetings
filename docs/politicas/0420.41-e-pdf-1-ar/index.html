Supervisión de escuelas Charter

Requisitos Para las Escuelas Charter Las escuelas Charter estarán sujetas a los términos de sus charters, cualquier memorándum de entendimiento con su autoridad otorgante, y otros requisitos legales que sean expresamente aplicables a las escuelas Charter, incluyendo, entre otros, los requisitos de que cada escuela Charter: 1. Sea no sectaria en sus programas, políticas de admisión, prácticas de empleo y todas las demás operaciones (Education Code 47605) 2. No discrimine a ningún estudiante por las características enumeradas en Education Code 220 (Education Code 47605) 3. No cobre colegiatura (Education Code 47605) 4. No cobre cuotas estudiantiles por ninguna actividad que sea un componente integral del programa educativo, excepto según lo autorizado por las disposiciones del Education Code que se aplican expresamente a las escuelas Charter 5. Se adhiera a todas las leyes que establecen la edad mínima para la asistencia escolar en escuelas públicas (Education Code 47610) 6. Atienda a estudiantes que sean residentes de California y que, si tienen más de 19 años de edad, estén matriculados de manera continua en una escuela pública y estén logrando un "progreso satisfactorio" hacia un diploma de preparatoria según se define en 5 CCR 11965 (Education Code 47612) 7. Atienda a estudiantes con discapacidades de la misma manera en que dichos estudiantes son atendidos en otras escuelas públicas (Education Code 47646, 56145) 8. Admita a todos los estudiantes que deseen asistir a la escuela, de acuerdo con los siguientes criterios y procedimientos: a. La admisión a la escuela Charter no se determinará según el lugar de residencia del estudiante, ni el de sus padres/tutores, dentro del estado, excepto que cualquier escuela pública existente que se convierta parcial o totalmente en una escuela Charter deberá adoptar y mantener una política que otorgue preferencia de admisión a los estudiantes que residan dentro del área de asistencia anterior de la escuela. (Education Code 47605) Si una escuela Charter estará ubicada físicamente en un área de asistencia de una escuela primaria pública en la que el 50 por ciento o más de la matrícula estudiantil sea elegible para comidas gratuitas o a precio reducido, también podrá establecer una preferencia de admisión para los estudiantes que estén matriculados actualmente en la escuela primaria pública y para los estudiantes que residan en el área de asistencia de la escuela pública. (Education Code 47605.3) b. Si el número de estudiantes que desean asistir a la escuela Charter supera la capacidad de la escuela, la asistencia, excepto para los estudiantes actuales de la escuela Charter, se determinará mediante un sorteo público al azar. Se extenderá preferencia a los estudiantes que asistan actualmente a la escuela Charter y a los estudiantes que residan en el distrito, excepto según lo dispuesto en Education Code 47614.5. (Education Code 47605) c. El distrito otorgante podrá permitir otras preferencias de admisión de manera individual para cada escuela, de conformidad con la ley. (Education Code 47605) 9. Matricule de inmediato a un estudiante sin hogar, excepto cuando dicha matrícula entre en conflicto con Education Code 47605(d) (Education Code 48850; 42 USC 11431-11435) 10. Cumpla con los requisitos de Education Code 48850-48859 en relación con la matrícula y la ubicación de jóvenes en hogares de crianza (Education Code 48853.5, 48859) 11. Si la escuela ofrece un programa de kínder: (Education Code 48000) a. Ofrezca un programa de kínder de transición (TK) a los estudiantes cuyo quinto cumpleaños sea entre el 2 de septiembre y el 2 de diciembre b. Garantice que cualquier maestro con credencial asignado por primera vez a impartir una clase de TK después del 1 de julio de 2015 cumpla con los requisitos establecidos en Education Code 48000 antes del 1 de agosto de 2020 12. Exija a sus maestros que posean un certificado, permiso u otro documento emitido por la Commission on Teacher Credentialing (CTC) equivalente al que se le requeriría a un maestro en otras escuelas públicas (Education Code 47605) 13. Proporcione capacitación anual sobre los requisitos de reporte de abuso y negligencia infantil a los empleados y personas que trabajen en nombre de la escuela que sean reporteros obligatorios, dentro de las primeras seis semanas de cada año escolar o dentro de las seis semanas posteriores al inicio del empleo (Education Code 44691) 14. No contrate a ninguna persona que haya sido condenada por un delito grave violento o grave, excepto según lo dispuesto por la ley, y, si la escuela celebra un contrato con una entidad para servicios específicos, verifique que cualquier empleado de dicha entidad que tendrá contacto con estudiantes haya pasado una verificación de antecedentes penales (Education Code 44830.1, 45122.1, 45125.1) 15. Reporte a la CTC cualquier cambio en el estatus de empleo de un empleado certificado (despido, no reelección, renuncia, suspensión, licencia administrativa sin pago por más de 10 días, jubilación u otra decisión de no emplear o no reemplear) como resultado de una acusación de conducta indebida o mientras una acusación de conducta indebida esté pendiente (Education Code 44030.5) 16. Cumpla con los requisitos de Education Code 47611 en relación con el State Teachers' Retirement System (Education Code 47610) 17. Cumpla con los requisitos de Government Code 3540-3549.3 relacionados con la negociación colectiva en el empleo en la educación pública (Education Code 47611.5) 18. Si la escuela atiende a estudiantes en los grados 7-12, adopte una política sobre prevención, intervención y postvención del suicidio con componentes específicos (Education Code 215) 19. Si la escuela atiende a estudiantes en el grado 9, adopte una política de ubicación en matemáticas que sea justa, objetiva y transparente con componentes específicos (Education Code 51224.7) 20. Cumpla con todos los estándares estatales y realice todas las evaluaciones estatales aplicables a las escuelas públicas no Charter (Education Code 47605, 47612.5, 60605, 60850-60859) 21. Hasta el 31 de julio de 2018, otorgue un diploma de preparatoria a cualquier estudiante que haya completado el grado 12 en el año escolar 2003-04 o en un año escolar posterior y que haya cumplido con todos los requisitos de graduación aplicables, excepto la aprobación del examen de salida de preparatoria (Education Code 60851.6) 22. Ofrezca al menos el número de minutos de instrucción requeridos por la ley para los niveles de grado que ofrece la escuela Charter (Education Code 47612.5) 23. Si la escuela ofrece estudio independiente, cumpla con los requisitos de Education Code 51745-51749.3, excepto que a la escuela se le podrá permitir ofrecer cursos requeridos para la graduación únicamente a través de estudio independiente como excepción a Education Code 51745(e) (Education Code 47612.5, 51747.3; 5 CCR 11705) 24. Identifique y reporte al Superintendent of Public Instruction (SPI) cualquier parte de su asistencia diaria promedio que se genere a través de instrucción fuera del salón de clases, incluyendo, entre otros, estudio independiente, estudio en casa, estudio laboral, y educación a distancia y mediante computadora (Education Code 47612.5, 47634.2; 5 CCR 11963.2) 25. Si la escuela ofrece atletismo competitivo, publique anualmente en el sitio web de la escuela o en el sitio web del operador del Charter la matrícula total de la escuela clasificada por género, el número de estudiantes que participan en atletismo competitivo clasificados por género, y el número de equipos de niños y niñas clasificados por deporte y por nivel de competencia (Education Code 221.9) 26. Si la escuela ofrece un programa atlético, proporcione anualmente hojas informativas sobre conmociones cerebrales/lesiones en la cabeza y paro cardíaco repentino a los atletas y a sus padres/tutores, las cuales deberán ser firmadas y devueltas a la escuela antes de que el atleta inicie los entrenamientos o la competencia. En caso de que se sospeche que un atleta sufrió una conmoción cerebral o lesión en la cabeza, pierde el conocimiento o se desmaya durante o inmediatamente después de participar en una actividad atlética, deberá ser retirado de inmediato de la actividad por el resto del día y no se le permitirá regresar a la actividad hasta que sea evaluado por un proveedor de atención médica con licencia y reciba autorización por escrito para hacerlo. (Education Code 33479-33479.5, 49475) 27. Consulte regularmente con los padres/tutores y maestros respecto a los programas educativos de la escuela (Education Code 47605) 28. Otorgue a los estudiantes el derecho a ejercer la libertad de expresión y de prensa incluyendo, entre otros, el uso de tableros de anuncios; la distribución de materiales impresos o peticiones; el uso de botones, insignias y otros distintivos; y el derecho de expresión en publicaciones oficiales (Education Code 48907, 48950) 29. Mantenga registros escritos y contemporáneos que documenten toda la asistencia de los estudiantes y ponga estos registros a disposición para auditoría e inspección (Education Code 47612.5) 30. Si un estudiante sujeto a educación obligatoria de tiempo completo es expulsado o abandona la escuela Charter sin graduarse o sin completar el año escolar por cualquier motivo, notifique al Superintendente del distrito escolar la última dirección conocida del estudiante dentro de los 30 días y, a solicitud, proporcione a dicho distrito una copia del expediente acumulativo del estudiante, incluyendo un historial de calificaciones o boleta de calificaciones, e información de salud (Education Code 47605) 31. Si la escuela atiende a estudiantes de preparatoria, presente ante la Student Aid Commission, para uso en el programa Cal Grant, el promedio de calificaciones (GPA) de todos los estudiantes en el grado 12 y la verificación de la graduación de preparatoria o su equivalente para los estudiantes que se graduaron en el año académico anterior. Sin embargo, dicha información no se presentará cuando los estudiantes opten por no participar o cuando las reglas de la Student Aid Commission les permitan proporcionar puntajes de exámenes en lugar del GPA. (Education Code 69432.9, 69432.92) 32. Desarrolle un plan de seguridad de transporte que incluya procedimientos para garantizar que un estudiante no quede desatendido en un autobús escolar, autobús de actividades estudiantiles, autobús para jóvenes o vehículo de transporte de cuidado infantil, y procedimientos y estándares para designar a un acompañante adulto, distinto del conductor, para acompañar a los estudiantes en un autobús de actividades escolares (Education Code 39831.3) 33. Cumpla con el California Building Standards Code según sea adoptado y aplicado por la agencia local de aplicación de normas de construcción con jurisdicción sobre el área en la que esté ubicada la escuela Charter, a menos que las instalaciones de la escuela Charter cumplan con cualquiera de las siguientes condiciones: (Education Code 47610, 47610.5) a. Las instalaciones cumplen con el Field Act conforme a Education Code 17280-17317 y 17365-17374. b. Las instalaciones son propiedad exclusiva o están bajo el control exclusivo de una entidad que no está sujeta al California Building Standards Code, incluyendo, entre otros, el gobierno federal. 34. Proporcione acomodaciones razonables en el plantel a una estudiante en período de lactancia para extraer leche materna, amamantar a un niño de pecho u atender otras necesidades relacionadas con la lactancia (Education Code 222) 35. Garantice la disponibilidad y el uso adecuado de auto-inyectores de epinefrina de emergencia mediante: (Education Code 49414) a. Proporcionar a las enfermeras escolares u otro personal voluntario capacitado al menos un dispositivo regular y uno de tamaño junior para escuelas primarias y, para escuelas secundarias, un dispositivo regular si no hay estudiantes que requieran un dispositivo de tamaño junior b. Distribuir un aviso al menos una vez por año escolar a todo el personal solicitando voluntarios y describiendo la capacitación que recibirán los voluntarios c. Proporcionar defensa e indemnización a los voluntarios por toda responsabilidad civil derivada de dicha administración 36. Si la escuela opta por poner a disposición un antagonista de opioides para personas que sufran, o de quienes se crea razonablemente que sufren, una sobredosis de opioides, cumpla con los requisitos de Education Code 49414.3, incluyendo, entre otros, proporcionar capacitación al personal que se ofrezca como voluntario para administrar el antagonista de opioides 37. Responda de manera oportuna a todas las consultas razonables del distrito, la oficina del condado de educación o el SPI, incluyendo, entre otros, las consultas relacionadas con los registros financieros de la escuela (Education Code 47604.3) 38. Prepare y presente anualmente informes financieros a la Mesa Directiva de educación del distrito y al Superintendente del condado de escuelas de acuerdo con el siguiente ciclo de informes: a. A más tardar el 1 de julio, un presupuesto preliminar para el año fiscal en curso. Para una escuela Charter en su primer año de operación, los estados financieros presentados con la petición del charter conforme a Education Code 47605(g) cumplirán con este requisito. (Education Code 47604.33) b. A más tardar el 1 de julio de cada año, una actualización de las metas de la escuela y las acciones para alcanzar dichas metas según se identifica en el charter, desarrollada utilizando la plantilla del plan de control local y rendición de cuentas adoptada por la State Board of Education. Este informe deberá incluir una revisión del avance hacia las metas, una evaluación de la efectividad de las acciones específicas para alcanzar las metas, una descripción de los cambios que la escuela realizará a las acciones específicas como resultado de la revisión y evaluación, y un listado y descripción de los gastos del año fiscal para implementar las acciones específicas. (Education Code 47604.33, 47606.5, 52064) Al realizar esta revisión, el órgano de gobierno de la escuela podrá considerar información cualitativa incluyendo, entre otros, los hallazgos que resulten de cualquier revisión de la calidad escolar realizada conforme a Education Code 52052 o cualquier otra revisión. En la medida en que sea posible, los datos se reportarán de manera consistente con la forma en que se reporta la información en una boleta de calificaciones de rendición de cuentas escolar. La actualización deberá desarrollarse en consulta con maestros, directores, administradores, otro personal escolar, padres/tutores y estudiantes. (Education Code 47606.5) c. A más tardar el 15 de diciembre, un informe financiero provisional para el año fiscal en curso que refleje los cambios hasta el 31 de octubre. (Education Code 47604.33) d. A más tardar el 15 de marzo, un segundo informe financiero provisional para el año fiscal en curso que refleje los cambios hasta el 31 de enero. (Education Code 47604.33) e. A más tardar el 15 de septiembre, un informe final no auditado del año anterior completo. El informe presentado a la Mesa Directiva deberá incluir un estado anual de todos los ingresos y gastos de la escuela Charter correspondientes al año fiscal anterior. (Education Code 42100, 47604.33) f. A más tardar el 15 de diciembre, una copia del informe de auditoría financiera independiente anual de la escuela Charter correspondiente al año fiscal anterior, a menos que la auditoría de la escuela Charter esté comprendida en la auditoría del distrito. El informe de auditoría también deberá presentarse ante el State Controller y el California Department of Education. (Education Code 47605) 39. Publique información específica relacionada con la prohibición de discriminación bajo Title IX of the Education Amendments of 1972 en un lugar destacado y visible en el sitio web de la escuela o en el sitio web del operador del Charter (Education Code 221.61) 40. Si es una escuela Charter con financiamiento directo, adopte e implemente Procedimientos uniformes de quejas para resolver quejas de discriminación ilegal o presunta violación de una ley o reglamento estatal o federal que rija los programas educativos, de conformidad con 5 CCR 4600-4670 (5 CCR 4600) 41. Adopte anualmente una boleta de calificaciones de rendición de cuentas escolar (Education Code 47612; California Constitution, Article 16, Section 8.5) Además, las escuelas Charter deberán cumplir con las constituciones estatal y federal, las leyes federales aplicables y las leyes estatales que se aplican a las agencias gubernamentales en general, tales como los requisitos del Brown Act establecidos en Government Code 54950-54963 y las leyes sobre conflicto de intereses en Government Code 1090-1099 y 87100-91014.
