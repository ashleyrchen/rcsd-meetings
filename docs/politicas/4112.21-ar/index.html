Programa de Pasantías Universitarias Las personas empleadas a través de un programa de pasantías universitarias están autorizadas para brindar el mismo servicio en los mismos niveles que lo autoriza una credencial regular. (Education Code 44454) El Superintendente o su designado puede celebrar un acuerdo con una institución de educación superior para el empleo de miembros del personal competentes y calificados de dicha institución, con el fin de supervisar y orientar a los pasantes mientras cumplen con sus responsabilidades en el distrito. (Education Code 44461) El Superintendente o su designado buscará la asistencia de una institución de educación superior para coordinar el programa de cada pasante. (Education Code 44465) Antes de que un pasante se inscriba en cualquier programa de una institución de educación superior para renovar su credencial de pasantía, el Superintendente o su designado deberá asesorar al pasante y planificar un programa para la primera renovación y las posteriores. (Education Code 44457) Programa de Pasantías del Distrito Las personas empleadas a través del programa de pasantías del distrito pueden ser asignadas para enseñar en cualquier clase de los grados K-12, en clases de educación bilingüe, o en programas de educación especial para estudiantes con discapacidades leves y moderadas. (Education Code 44830.3) Cada pasante del distrito recibirá asistencia y orientación de cualquiera de los siguientes: (Education Code 44326, 44830.3) - Un empleado certificado que posea una certificación válida en el mismo nivel o el mismo tipo de credencial que el pasante del distrito al que sirve, y que sea seleccionado mediante un proceso competitivo adoptado por la Mesa Directiva de Educación después de consultar con la unidad representativa exclusiva de maestros - Personal empleado por una institución de educación superior para supervisar a maestros estudiantes El Superintendente o su designado deberá, en consulta con una institución de educación superior, desarrollar e implementar un plan de desarrollo profesional para los pasantes del distrito. Este plan deberá incluir, pero no necesariamente limitarse a, los componentes descritos en Education Code 44830.3, incluyendo disposiciones para la capacitación previa al servicio obligatoria adaptada al nivel de grado o clase que se enseñará, instrucción adicional durante el primer año de servicio cuando se requiera, y una evaluación anual del pasante del distrito. (Education Code 44830.3) El Superintendente o su designado deberá asegurarse de que se mantengan los registros apropiados que muestren el crédito obtenido por cada pasante del distrito. (Education Code 44327) Cuando la credencial de un pasante del distrito venza, el Superintendente o su designado puede recomendar a la Commission on Teacher Credentialing (CTC) que la credencial del pasante sea extendida por un año. (Education Code 44325) Cuando un pasante del distrito haya completado satisfactoriamente el servicio suficiente para cumplir con los estándares del programa y las evaluaciones de desempeño, la Mesa Directiva puede recomendar a la CTC que al pasante se le otorgue una credencial preliminar. La Mesa Directiva puede recomendar la emisión de una credencial profesional clara si el programa incluye un plan de inducción aprobado o cursos avanzados aprobados por la CTC que cumplan con los requisitos de Education Code 44259(c) para una credencial profesional clara. (Education Code 44328, 44468, 44830.3) Opción de Finalización Anticipada Un pasante puede elegir una opción de finalización anticipada que conduzca a una credencial preliminar de cinco años al completar los siguientes requisitos: (Education Code 44468) - Aprobar la evaluación de conocimientos de fundamentos de la enseñanza aprobada por la CTC - Aprobar la evaluación de desempeño docente aprobada por la CTC - El pasante deberá primero aprobar la evaluación de fundamentos de la enseñanza descrita en el punto #1 antes de calificar para tomar la evaluación de desempeño docente. - El pasante puede tomar la evaluación de desempeño docente solo una vez como parte de la opción de finalización anticipada. Un pasante que no tenga éxito en esta evaluación puede completar su programa de pasantía y sus calificaciones serán utilizadas para elaborar un plan de desarrollo profesional individualizado que enfatice la preparación en áreas donde se justifica un crecimiento adicional y que exima la preparación en áreas donde el pasante ha demostrado competencia. El pasante deberá volver a tomar y aprobar la evaluación de desempeño docente al final de la pasantía para ser considerado para una recomendación ante la CTC. - Para los pasantes que persiguen una credencial preliminar de materias múltiples, aprobar la evaluación de competencia en instrucción de lectura descrita en Education Code 44283 - Cumplir con los requisitos de aptitud para maestros establecidos en Education Code 44339, 44340, y 44341
